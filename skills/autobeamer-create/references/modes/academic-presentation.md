# Academic-Presentation Mode

Use `academic-presentation` for live talks, seminars, defenses, journal clubs, and academic sharing.

## Learning Philosophy

The deck supports a speaker in a time-bounded room. It should make the audience care quickly, communicate the contribution with enough evidence to be credible, and leave a clear path for questions.

> **English decks:** also load [../writing/en-technical-style.md](../writing/en-technical-style.md) —
> payoff-first titles, one relationship per line, plain register, captions/equations as
> evidence. Telegraphic here, but each line still advances and lands its point on the stress
> position.

## Authoring Rules

- Optimize for audience, time, and one recoverable narrative thread.
- Use telegraphic slide text except where one framing sentence improves comprehension.
- Allocate time explicitly: motivation, background, core result, evidence, limitations, and closing.
- Keep proofs and implementation detail to the level the talk format can support; move extra rigor to backup.
- Include a references slide as the second-to-last slide before Thank You.
- Add backup slides after Thank You for expected questions.

## Acceptance Gates

- The structure plan names the target audience and timing budget.
- The deck has a clear opening motivation and a non-empty closing takeaway.
- The slide count is compatible with the requested talk duration.
- Every core claim has evidence, citation, or backup support.
